# l10n_bg — Bulgarian Localization for Odoo 18 & 19

**Live:** https://rosenvladimirov.github.io/l10n-bg/

Single-file bilingual (BG/EN) **documentation explorer** — the full README
of every module across the 5-repo Bulgarian-localization ecosystem
(~115 modules: accounting, payroll, banking, NRA/NSI/NOI, fiscal printers,
payments, AI), searchable, all in one page. No build step.

- `index.html` — generated docs explorer (all READMEs embedded, pastel-blue)
- `L10N_BG_ECOSYSTEM.md` — cross-repo overview (source of truth)

Maintainer: **Rosen Vladimirov** — Bulgarian Odoo developer & consultant,
11+ years on l10n_bg · [github.com/rosenvladimirov](https://github.com/rosenvladimirov)
